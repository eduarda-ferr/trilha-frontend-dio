Introdução

voltar

A introdução é baseada na estrutura básica em um documento HTML composto com as Tags principais como dentro dela, suas demais como <html>, <head>, <title>, <body> e elementos de conteúdo como <header>.

  • Estrutura básica: <html> > <head> > <title> > <body> > <header>, entre outras.
  • Títulos: <h1> até <h6> — representam títulos de diferentes níveis e tamanhos.
  • Formatação de texto:
    • <i>: deixa o texto em itálico.
    • <strong>: deixa o texto em negrito.
    • <mark>: destaca o texto como um marcador.
    • <sup>: posiciona o texto como sobrescrito, exemplo: x2.
    • <sub>: posiciona o texto como subscrito, exemplo: H2O.
    • <blockquote>: insere uma citação em bloco, como:
      “O conhecimento é poder.”

Formulário

voltar

Nesta seção, aprofunda-se nos tipos de entrada em HTML, como base para criação de interfaces de cadastro de pesquisa.

Tipos de entrada (<input>)
  • text: campo de texto simples
  • number: campo numérico
  • password: campo de senha (caracteres ocultos)
  • email: valida e coleta e-mails
  • url: campo para links
  • radio: seleção única entre várias opções
  • checkbox: seleção múltipla
  • color: seletor de cores
  • range: controle deslizante
  • week, month: seleção de semana e mês
Elementos e atributos do <form>
  • action: define para onde os dados serão enviados
  • method: define o método de envio (GET ou POST)
  • name: nome identificador do formulário
  • target: define onde o resultado será exibido (por exemplo, _blank)
  • onsubmit: executa uma ação ao enviar o formulário (como alertas)
  • autocomplete: ativa/desativa sugestões de preenchimento automático
Outros elementos
  • <select>: menu suspenso
  • <textarea>: área para textos longos
  • <button>: botão personalizável com os tipos submit, button e reset
  • min / max: definem limites de valor
  • disabled: desativa o campo
  • name[]: permite enviar múltiplos valores com o mesmo nome (útil em arrays)
  • selected: define a opção padrão selecionada em <select>

Formatação

voltar

Proseguimos agora para a área de formatação de texto e organização de conteúdo.

Tags principais de formatação
  • <i>: deixa o texto em itálico.
  • <b>: aplica negrito simples.
  • <strong>: dá ênfase semântica, indicando importância.
  • <u>: adiciona sublinhado ao texto.
  • <mark>: destaca com marca-texto.
  • <blockquote>: insere uma citação em bloco:
    “Design é inteligência feita visível.”
  • <font>: (Deve ser evitada em projetos modernos.)
  • <div> é usada para agrupar blocos de conteúdo e facilitar a organização da página.
  • <span> é usada para aplicar estilo em trechos específicos de texto inline.
Novas tags e funcionalidades
  • <fieldset>: agrupa elementos de um formulário.
  • <legend>: fornece uma legenda para o <fieldset>.
  • <label>: associa texto a um campo de formulário.
  • <embed>: incorpora conteúdo multimídia (como vídeos ou PDFs).
  • <iframe>: exibe outra página HTML dentro da atual.
Elementos CSS
  • margin: espaçamento externo do elemento.
  • padding: espaçamento interno entre conteúdo e borda.
  • border-radius: arredonda os cantos das bordas.
  • background-color: define a cor de fundo do elemento.

Tabelas

Voltar

Esta seção aborda a criação e estruturação de tabelas, incluindo semântica e estilização com CSS.

Estrutura semântica

A tabela está dividida em três partes:

  • <thead>: cabeçalho, onde os títulos das colunas são definidos com <th>.
  • <tbody>: corpo, com os dados organizados em <tr> (linhas) e <td> (colunas).
  • <tfoot>: rodapé, usado para resumos ou totais.
Tags e atributos
  • <caption>: nome da tabela.
  • colspan="2": uma célula ocupa duas colunas.
  • title: mostra uma dica ao passar o mouse sobre o título.
  • cellspacing e cellpadding: controlam espaçamento.
  • align e width: alinhamento e largura.
Elementos CSS
  • Sombra (box-shadow): adiciona profundidade visual à tabela.
  • Estilo para <thead> e <tfoot>: cores de fundo diferenciadas.
  • Zebragem: nth-child(even) destaca linhas pares.
  • Hover: altera o fundo ao passar o mouse (:hover) e muda o cursor.

Mídia

Voltar

Exploramos como inserir imagens, áudios e vídeos usando HTML5.

Adição de imagens com <img>

É possível adicionar imagens de diferentes formatos, como .gif, .png, .webp e .jpg.

  • src: caminho da imagem
  • alt: descrição alternativa (essencial para acessibilidade e SEO)
  • title: dica exibida ao passar o mouse
Áudio com <audio>

Para inserir áudio, usamos a tag <audio> com uma ou mais fontes <source>.

  • Formatos recomendados: .mp3, .ogg, .wav
  • O formato .mkv não é compatível com áudio em HTML5
Vídeo com <video>

Vídeos podem ser incorporados usando a tag <video> com controles e trilhas de legenda.

  • controls: exibe os controles de reprodução
  • <source>: define o arquivo de vídeo
  • <track>: adiciona legenda ao vídeo (formato .vtt)
Vídeos incorporados com <iframe>

É possível incorporar vídeos de plataformas como exemplo, YouTube, usando <iframe>.

  • src: link do vídeo
  • allowfullscreen: permite tela cheia
  • referrerpolicy e allow: aumentam segurança e funcionalidades

Semântica

Voltar

HTML semântico consiste o uso de tags com significado próprio, melhorando a organização do código.

Estruturação
  • <header>: define o cabeçalho da página ou seção.
  • <nav>: agrupa links de navegação.
  • <main>: representa o conteúdo principal da página (não deve estar dentro de header ou footer).
  • <article>: representa um conteúdo independente e completo, como uma notícia ou post.
  • <section>: agrupa conteúdos relacionados.
  • <aside>: conteúdo complementar (ex: curiosidades, barras laterais).
  • <footer>: rodapé com informações finais, como <address>.
Imagens e Ilustrações
  • <figure>: agrupa uma imagem com sua legenda.
  • <figcaption>: legenda explicativa da imagem.
  • <picture> e <source>: permitem carregar diferentes imagens dependendo da largura da tela (responsividade).
  • alt: atributo obrigatório para acessibilidade, descrevendo o conteúdo da imagem.
Estilização CSS
  • display: flex no nav: alinha os itens da navegação em linha.
  • :hover nos links: altera a cor e o fundo ao passar o mouse.
  • blockquote com border-left, background-color e color: estilo visual para citações destacadas.
Outras Tags Semânticas
<del> — Texto Removido

Exemplo:
<p>Preço original: <del>R$ 100,00</del> R$ 80,00</p>
Resultado:

Preço original: R$ 100,00 R$ 80,00

<abbr> — Abreviação

Usada para indicar siglas ou abreviações. O atributo title mostra o significado completo ao passar o mouse.
